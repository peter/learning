## Hello World

Java:

```java
public class Hello {
  public static void main(String[] args) {
    System.out.println("Hello World!");
  }
}
```

Python:

```python
print('Hello World!')
```

Clojure:

```clojure
(println "Hello World!")
```

Elixir:

```elixir
IO.puts "Hello World"
```

JavaScript:

```javascript
console.log('Hello World!')
```
